# Zeller Frontend Challenge

## Vercel deployment Link

```bash
https://zeller-frontend-challenge.vercel.app/
```

## To run app at localhost, Clone the repository to your local machine:

```bash
git clone https://github.com/RushiShirkar/zeller-frontend-challenge.git
cd zeller-frontend-challenge
```

## Install

Install the dependencies:

```bash
npm install
```

## Configuration

Create a `.env` file in the root directory and add the following environment variables:

```env
REACT_APP_GRAPHQL_API_KEY=
REACT_APP_GRAPHQL_API_URL=
REACT_APP_AWS_REGION=
```

## Running the Application

Start the development server:

```bash
npm start
```

## Testing

### Unit Tests

Run the unit tests:

```bash
npm run test
```

### E2E Tests

Run the Cypress end-to-end tests:

```bash
npm run cypress
```
